Sistema de Vendas Online (MVC)

Plataforma web de E-commerce com Painel Administrativo integrado, projetada para gerenciar vendas e produtos de forma automatizada. O sistema conta com controle de acesso multinível (Admin/Cliente), tela de login, cadastro com validação de e-mails duplicados e gerenciamento completo de produtos (CRUD) com suporte a upload de imagens. Na interface do cliente, a vitrine calcula dinamicamente a conversão de moedas (Dólar para Real) direto na view com Thymeleaf. Desenvolvido com Spring Boot, HTML5 e CSS3, focando na experiência do usuário e na eficiência de processos.

Interface Dinâmica com Bootstrap

Web app interativo projetado como um painel de informações sobre o elenco do Corinthians. A aplicação demonstra domínio em ferramentas de front-end ao combinar a grade responsiva do Bootstrap com elementos visuais sofisticados, como efeitos de sombreamento (shadow) e gradientes de fundo complexos. A arquitetura da página permite o isolamento de dados por meio de modais acionados por seletores de ID específicos. O resultado é um produto web limpo, altamente intuitivo e otimizado para navegação em dispositivos móveis e desktops.

Dashboard Responsiva com Grid System

Protótipo de interface de controle (Dashboard) desenvolvido como parte estrutural de um Trabalho de Conclusão de Curso (TCC). O projeto foca no domínio técnico do Grid System do Bootstrap 5, utilizando pontos de quebra inteligentes (col-12, col-md-6, col-lg-4) para garantir uma adaptação perfeita do layout entre smartphones, tablets e desktops. Conta com uma distribuição modular em cartões funcionais estruturados com classes de utilitários nativos e gutters (g-4) para espaçamento fluido. Uma solução limpa que demonstra capacidade em criar front-ends organizados, simétricos e com foco total na usabilidade cross-device.

Protótipo Didático de Grid e Flexbox

Projeto focado no estudo aprofundado de arquitetura front-end e comportamento de layouts modernos baseados em Flexbox e tabelas virtuais do Bootstrap 5. A página funciona como um mostruário técnico que exemplifica a matemática visual por trás do design responsivo, apresentando exemplos práticos de frações comuns (1/12, 1/3, 1/2 e largura total). A estrutura destaca o uso correto de seletores filhos em folhas de estilo locais para depuração visual de componentes de interface. Uma solução limpa, direta e puramente focada em boas práticas de engenharia de software voltada para a camada de apresentação visual.

Painel de Customização de UI em React

Um componente interativo que demonstra o poder da renderização condicional e da manipulação de propriedades dinâmicas combinando React e CSS3. O sistema gerencia múltiplos estados independentes para controlar a estilização de um avatar e do container principal, utilizando template literals para injetar dinamicamente classes conforme as escolhas do usuário. O projeto apresenta transições suaves de layout (transition: .5s) ao alternar esquemas de cores e dimensões, provando excelente domínio em animações de interface sem comprometer a performance. Uma interface intuitiva baseada em conceitos de microrreatividade e estados locais.

Vitrine Digital Responsiva em React

Aplicação front-end estruturada em React focada na criação de uma interface de vitrine comercial fluida e modular. O projeto adota a arquitetura de CSS Modules para encapsular a estilização de cada componente (Cabecalho, Footer e CardsVitrineA), prevenindo vazamentos de escopo no CSS global. A disposição dos elementos é gerenciada por meio de Flexbox dinâmico, adaptando o grid de produtos automaticamente para telas menores com o uso estratégico de Media Queries. Uma solução robusta voltada para e-commerce que demonstra domínio em componentização estruturada e design responsivo.

Tela de Login Responsiva com HTML5 e CSS3

Interface de autenticação (Screen Login) desenvolvida puramente com HTML5 estrutural e CSS3 avançado, sem o uso de frameworks externos. O projeto destaca a criação de formulários acessíveis e semânticos, utilizando elementos nativos para captura de credenciais de forma segura. A estilização aplica conceitos modernos de design visual, incluindo centralização perfeita de elementos na tela com Flexbox, uso de variáveis CSS para paleta de cores, cantos arredondados e transições suaves de estado (hover) nos campos de entrada e botões. Uma solução leve, focada em performance e fidelidade visual.

Web Design Responsivo e Arquitetura de Layout Nativo

Interface web desenvolvida puramente com tecnologias nativas, projetada com foco total na legibilidade, tipografia e design clean. O layout implementa técnicas fundamentais de responsividade e adaptação de elementos na tela, utilizando uma folha de estilos externa para gerenciar margens, espaçamentos e fontes personalizadas via Google Fonts. O grande diferencial técnico está no uso de seletores e pseudoclasses para enriquecer a experiência do usuário com microinterações em menus e links. Uma excelente demonstração de domínio técnico sobre o comportamento físico de elementos HTML e estilização corporativa.
